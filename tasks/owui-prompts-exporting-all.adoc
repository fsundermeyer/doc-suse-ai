[[owui-prompts-exporting-all]]
= Exporting all existing system prompts

This procedure shows how to export all existing system prompts. 

.Requirements
** {empty}
+

include::../snippets/openwebui-requirement-admin-privileges.adoc[leveloffset=1]
. In the {owui} interface, click menu:Workspace[] in the upper part of the left panel and select menu:Prompts[] . 
. Click menu:Export Prompts[] in the bottom right of the prompt list. 
. Enter the path to the JSON file that will contain the exported prompts and confirm with menu:Save[] .
