[[openwebui-user-role-assigning]]
= Assigning roles to existing users in {owui}

This topic explains how to assign one of the available roles to an existing user account.
To assign the role when creating a new user account, refer to <<_openwebui_user_account_creating>>. 

.Requirements

include::../snippets/openwebui-requirement-admin-privileges.adoc[leveloffset=+1]
. In the bottom left of the {owui} window, click your avatar icon to open the user menu and select menu:Admin Panel[] . 
. On the menu:Users[] screen, scroll to the user whose role you want to modify. Use the pager at the bottom if the user is not listed on the current screen. 
+

.Assigning user role [[_openwebui_users_role_assigning]]

image::openwebui-users-role-assigning.png[Assigning user role,scaledwidth=75%]
. Click the role button to the left of the user name to cycle through "`pending`" -> "`user`" -> "`admin`" role.
