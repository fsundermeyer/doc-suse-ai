[#pytorch-installing]
= Installing {pytorch}

{pytorch} is a widely used open-source deep-learning framework that supports both CPU and GPU acceleration.
When deployed with the {suseaia} stack, the {pytorch} {helm} chart lets you inject your own training or inference code into the container and run it on {nvidia} GPUs available in your cluster.

[#pytorch-installing-app-details]
== Details about the {pytorch} application

Before deploying {pytorch}, it is important to know more about the supported configurations and documentation.
The following command provides the corresponding details:

[source]
----
helm show values oci://dp.apps.rancher.io/charts/pytorch
----

Alternatively, you can also refer to the link:https://apps.rancher.io/applications/pytorch[{pytorch} {helm} chart page].
It contains {pytorch} dependencies, available versions and the link to pull the {pytorch} container image.

[#pytorch-installing-procedure]
== {pytorch} installation procedure

include::../snippets/ai-library-requirement.adoc[]

. Create a `pytorch_custom_overrides.yaml` file to override the values of the parent {helm} chart.
Find examples of {pytorch} override files in xref:pytorch-helm-overrides[] and a list of all valid options and their values in xref:pytorch-helm-values[].
. Install the {pytorch} {helm} chart using the `pytorch_custom_overrides.yaml` file using the following command.
+
[source,subs="+attributes"]
----
{prompt_user}helm upgrade --install \
  pytorch oci://dp.apps.rancher.io/charts/pytorch \
  -n SUSE_AI_NAMESPACE \
  -f pytorch_custom_overrides.yaml
----

[#pytorch-upgrading]
== Upgrading {pytorch}

You can upgrade {pytorch} to a specific version by running the following command:

[source,subs="+attributes"]
----
{prompt_user}helm upgrade \
  pytorch oci://dp.apps.rancher.io/charts/pytorch \
  -n SUSE_AI_NAMESPACE \
  --version VERSION_NUMBER \
  -f pytorch_custom_overrides.yaml
----

[TIP]
====
If you omit the `--version` option, {pytorch} gets upgraded to the latest available version.
====

[#pytorch-uninstalling]
== Uninstalling {pytorch}

To uninstall {pytorch}, run the following command:

[source,subs="+attributes"]
----
{prompt_user}helm uninstall pytorch -n SUSE_AI_NAMESPACE
----
