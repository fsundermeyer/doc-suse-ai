= Selecting {mcpo} services from the chat input field

{mcp} servers offer additional services that enhance the AI model responses and make them more specific.
This section details how to enable individual {mcpo} services from the input field. 


.Requirements
* At least one {mcp} server must be preconfigured. Refer to https://documentation.suse.com/suse-ai/1.0/html/AI-deployment-intro/index.html#mcpo-installing[Installing {mcpo}] to learn how to configure one.
--
. Click the "+" sign to the left of the chat input field. 
. From the drop-down list that appears, select the {mcp} service that you want to include for the current chat. 
+
.Selecting {mcp} services selecting_mcp_services
image::owui-usage-mcpo.png[A screenshot of selecting {mcp} services from the chat input field,scaledwidth=50%]
. Enter a prompt that triggers a call to one of the selected {mcp} services. You should see the tool invocation listed below the response. 
+
.Response from an {mcp} service selecting_mcp_answer
image::owui-usage-mcpo-question.png[A screenshot of a response from an {mcp} service,scaledwidth=50%]
--

[TIP]
====
For more details about {mcpo} integration and usage with {owui}, refer to the https://https://docs.openwebui.com/openapi-servers/open-webui/[{owui} integration documentation]. 
====
